## 🚀 用户使用流程

### 流程1：快速部署（推荐）

```bash
# 1. 在云控制台配置防火墙/安全组（开放80, 443, 8080端口）

# 2. SSH连接到服务器

# 3. 安装Docker
curl -fsSL https://get.docker.com | bash

# 4. 克隆项目
git clone https://github.com/dctx479/UPS.git
cd UPS

# 5. 一键部署
./quick-start.sh

# 6. 配置公网访问
sudo ./setup-public-access.sh

# 完成！
```

总耗时：**约15-20分钟**

### 流程2：查阅专属指南

```
1. 查看 docs/CLOUD_DEPLOYMENT_INDEX.md
2. 根据云服务商选择对应指南
3. 按照指南逐步操作
4. 遇到问题查看故障排查章节
```
