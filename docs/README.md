## 🚀 快速预览

```bash
# 克隆项目
git clone <repository-url>

# 启动所有服务
docker-compose up -d

# 访问系统
http://localhost:8080
```

## 🏗️ 技术栈

- **后端**: Spring Boot 3.2 + Spring Cloud
- **数据库**: PostgreSQL + MongoDB + Redis
- **前端**: Flutter
- **基础设施**: Docker + Consul

## 📊 系统特性

✅ 微服务架构 - Consul服务发现 + API Gateway
✅ 高性能 - Redis缓存 + MongoDB索引优化
✅ 高可用 - 熔断降级 + 分布式事务补偿
✅ 安全可靠 - JWT认证 + API限流 + 审计日志
✅ 测试完备 - 93个测试用例，75%+覆盖率
